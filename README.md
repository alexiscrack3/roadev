## Prerequisites

- Ruby 2.7.0
- Rails 6.1.4
- Node.js 12.18.3+
- Yarn
- Docker

## Installation

Clone the repo and cd into the directory:

```bash
git clone https://github.com/alexiscrack3/roadev
```

```bash
cd roadev
```

```bash
bundle install
```

```bash
yarn install
```

### Usage

```bash
docker-compose up -d db
```
